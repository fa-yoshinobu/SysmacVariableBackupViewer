- Windows 10 or later
- .NET 6.0 Runtime
- Minimum 4GB RAM recommended

Usage:
This application is designed to read and display variable data from Sysmac Studio
XML backup files. It provides a user-friendly interface for viewing, filtering,
and exporting variable information.

- No modification of original XML files is performed
- Export functionality creates new files without affecting originals
